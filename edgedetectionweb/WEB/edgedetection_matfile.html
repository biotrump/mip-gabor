The elements of the result that is displayed in the output window are first rescaled in such a way that the minimum element has the value 0 and the maximum element has the value 255. This rescaled result is displayed in the output window. In some cases, such as texture analysis, you may need the original result that is not rescaled. You can download this result as a Matlab file or an ASCII file.

The matlab file

The matlab file can be loaded and displayed in Matlab (version 6.5 and higher) as follows:
  • Download the file. The file name generated by our program has the following structure: ml*****edgedetection.mat, where the asterisks represent digits.
  • To load the contents of the file in Matlab, put the file first in your Matlab working directory and then make the assignment: var = load('ml*****edgedetection.mat').
  • To format the result as an image make the following assignment: img = var.img;
  • This unscaled image can be displayed by the following command: image(img)
  • The colormap can be changed to grayscale by: colormap('gray')
  • To display a scaled version of the result image use the following command: imagesc(img)

The ascii file

The ascii file contains all elements of the the unscaled result image as plain text. This file can also be loaded and displayed in Matlab as follows:
  • Download the file. The file name generated by our program has the following structure: ml*****edgedetection.asc where the asterisks represent digits.
  • Put the file in your current Matlab working directory.
  • Load the unscaled image in Matlab as follows: img = load('ml*****edgedetection.asc')
  • To display the unscaled result image use the following command: image(img)
  • The colormap can be changed to grayscale by: colormap('gray')
  • To display a scaled version of the result image use the following command: imagesc(img)
